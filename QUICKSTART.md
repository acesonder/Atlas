# Quick Start Guide - Atlas Solar System Explorer

## 🚀 Getting Started in 3 Steps

### Step 1: Get the Files
```bash
git clone https://github.com/acesonder/Atlas.git
cd Atlas
```

### Step 2: Open in Browser
**Option A - Direct Open (Easiest)**
- Double-click `index.html`
- Your default browser will open the application

**Option B - Local Server (Recommended)**
```bash
python3 -m http.server 8000
```
Then open: `http://localhost:8000`

### Step 3: Start Exploring!
The solar system will load automatically.

## 🎮 Controls Cheat Sheet

### Camera
| Action | How To |
|--------|--------|
| Rotate View | Left Click + Drag |
| Pan View | Right Click + Drag |
| Zoom In/Out | Scroll Wheel |
| Reset View | Click "Reset Camera" |

### Time Travel
| Action | How To |
|--------|--------|
| Speed Up | Slide Time Scale to the right |
| Slow Down | Slide Time Scale to the left |
| Go Backwards | Slide Time Scale to negative values |
| Pause | Set Time Scale to 0 |
| Jump to Date | Use Date Picker |
| Return to Now | Click "Reset to Now" |

### Display Options
| Toggle | Effect |
|--------|--------|
| Show Planet Labels | Planet names appear |
| Show Moon Labels | Moon names appear |
| Show Star Labels | Sun label appears |
| Show Orbits | Orbital paths visible |
| Show Comets | Comets and tails appear |

## 🎯 Try These First!

1. **See the Whole Solar System**
   - Zoom out using scroll wheel
   - Rotate to see all planets

2. **Watch Planets Orbit**
   - Set time scale to 50x
   - Watch Earth complete its orbit around the Sun

3. **Find a Famous Comet**
   - Check "Show Comets"
   - Look for Halley's Comet with its tail

4. **Time Travel**
   - Pick a date in the past (your birthday!)
   - See where the planets were that day

5. **Examine Saturn's Rings**
   - Click and drag to rotate
   - Zoom in on Saturn to see the rings clearly

## ❓ Troubleshooting

**Can't see anything?**
- Check that JavaScript is enabled
- Ensure you have internet connection (for Three.js library)
- Try refreshing the page (F5)

**Running slow?**
- Close other browser tabs
- Uncheck "Show Comets" and "Show Orbits"
- Use a modern browser (Chrome recommended)

**Want to see the UI without loading 3D?**
- Open `demo.html` instead
- This shows the interface without requiring internet
