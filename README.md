# Atlas - Solar System Explorer 🌌

An interactive web application showcasing Comet ATLAS (C/2024 G3) and humanity's interplanetary spacecraft exploring our solar system and beyond.

## Features

- **Comet ATLAS Information**: Detailed information about Comet ATLAS (C/2024 G3), an interstellar visitor to our solar system
- **Spacecraft Database**: Comprehensive details about spacecraft and probes that have left or are leaving our solar system:
  - Voyager 1 & 2 (in interstellar space)
  - Pioneer 10 & 11 (beyond the solar system)
  - New Horizons (Kuiper Belt explorer)
  - Parker Solar Probe (studying the Sun)
- **Interactive Navigation**: Easy-to-use interface to explore different sections
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Educational Content**: Learn about space exploration and celestial objects

## Getting Started

Simply open `index.html` in your web browser to start exploring!

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/acesonder/Atlas.git
   cd Atlas
   ```

2. Open in browser:
   ```bash
   open index.html
   # or
   python3 -m http.server 8000
   # then navigate to http://localhost:8000
   ```
